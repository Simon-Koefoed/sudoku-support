Frequently Asked Questions

How do I play Sudoku?

Sudoku is a logic-based number placement puzzle. The objective is to fill a 9×9 grid with digits so that each column, each row, and each of the nine 3×3 subgrids contains all of the digits from 1 to 9.

In Sudoku Corner, simply tap on a cell to select it, then tap on a number to place it. If you're not sure about a number, you can use the draft mode to make notes.

What is Draft Mode?

Draft Mode allows you to make notes in cells when you're not sure which number belongs there. To use Draft Mode:

  1. Tap the pencil icon to enable Draft Mode
  2. Select a cell
  3. Tap numbers to add them as notes
  4. Tap a number again to remove it from your notes

Draft Mode is a great way to keep track of possible numbers for each cell as you solve the puzzle.

How do difficulty levels work?

Sudoku Corner offers three difficulty levels:

  • Easy: More numbers are provided at the start, making the puzzle simpler to solve.
  • Medium: A balanced challenge with fewer starting numbers.
  • Hard: Fewer starting numbers and more complex logical steps required to solve.

Choose the difficulty level that matches your experience and skill level.

How do I track my progress?

Sudoku Corner keeps track of your game statistics, including:

  • Best completion time for each difficulty level
  • Average completion time
  • Best moves count
  • Average moves count
  • Total games played

You can view your statistics by tapping the Statistics button on the main menu.

How do I reset my statistics?

If you want to reset your game statistics:

  1. Go to the Statistics screen from the main menu
  2. Scroll to the bottom of the screen
  3. Tap the "Clear Statistics" button

Note that this action cannot be undone.
